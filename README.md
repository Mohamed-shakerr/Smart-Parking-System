 🅿️ Smart Parking System

A Windows desktop application built with **C# and .NET 8** to automate and streamline parking lot management operations.

---

 📌 Overview

Smart Parking System replaces manual parking management with a clean, efficient desktop solution. It handles everything from slot allocation to revenue tracking — in real time.

---

 ✨ Features

- 🔐 **Login System** — Secure access to the application
- 🚗 **Park a Car** — Auto-assign or manually select a parking slot
- 🚪 **Exit a Car** — Calculate duration and auto-generate parking fee
- 📊 **Dashboard** — Live view of available vs. occupied slots
- 👁️ **View Parking** — See all current cars with their details
- 📋 **Reports** — Full transaction history and daily revenue tracking
- 💾 **Data Persistence** — Parking and transaction data saved locally across sessions

---

🛠️ Tech Stack

| Technology | Usage |
